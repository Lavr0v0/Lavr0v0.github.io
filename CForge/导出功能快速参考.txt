├─────────────────────────────────────────────────────────────────┤

│ 尺寸: 128x128 像素                                               │

┌─────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│ 📄 TXT 格式                                                      │
│   • 纯文本，带边框装饰                                           │
│   • 属性显示为 ★☆ 星级                                          │
│   • 适合: 快速分享、打印、存档                                   │
│   • 文件名: [角色名]_档案_[时间戳].txt                           │
│                                                                  │
│ 📝 Markdown 格式                                                 │
│   • 标准 MD 格式                                                 │
│   • 照片以 Base64 嵌入                                           │
│   • 属性以表格展示                                               │
│   • 适合: 文档管理、团队协作、版本控制                           │
│   • 支持: GitHub, Notion, Obsidian                               │
│   • 文件名: [角色名]_档案_[时间戳].md                            │
│                                                                  │
│ 🖼️ PNG 图片                                                      │
│   • 高清截图（2x scale）                                         │
│   • 包含完整预览卡片                                             │
│   • 包含上传的照片                                               │
│   • 适合: 社交媒体、视觉展示                                     │
│   • 首次使用自动加载 html2canvas 库                              │
│   • 文件名: [角色名]_档案_[时间戳].png                           │
│                                                                  │
│ 📋 复制文本                                                      │
│   • 一键复制到剪贴板                                             │
│   • 与 TXT 格式相同                                              │
│   • 适合: 快速粘贴到聊天、邮件                                   │

┌─────────────────────────────────────────────────────────────────┐
│ 🔧 技术细节                                                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│ 照片存储:                                                        │
│   • Base64 格式存储在内存中                                      │
│   • 不上传到服务器                                               │
│   • 刷新页面后需要重新上传                                       │
│                                                                  │
│ 导出实现:                                                        │
│   • TXT/MD: Blob API                                             │
│   • PNG: html2canvas 库                                          │
│   • 复制: Clipboard API                                          │
│                                                                  │
│ 浏览器兼容:                                                      │
│   • Chrome 90+   ✅                                              │
│   • Firefox 88+  ✅                                              │
│   • Safari 14+   ✅                                              │
│   • Edge 90+     ✅                                              │
│                                                                  │
│ 性能:                                                            │
│   • 照片上传: < 100ms                                            │
│   • TXT 导出: < 50ms                                             │
│   • MD 导出:  < 100ms                                            │
│   • PNG 导出: < 2s（首次）                                       │
│   • 复制文本: < 50ms                                             │
│                                                                  │

┌─────────────────────────────────────────────────────────────────┐
│ ❓ 常见问题                                                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│ Q: 照片上传失败？                                                │
│ A: 检查图片格式和大小（< 1MB）                                   │
│                                                                  │
│ Q: PNG 导出失败？                                                │
│ A: 首次使用需要加载库，请等待几秒                                │
│                                                                  │
│ Q: Markdown 文件太大？                                           │
│ A: 照片以 Base64 编码，会增加文件大小（约 1.33 倍）              │
│                                                                  │
│ Q: 照片会上传到服务器吗？                                        │
│ A: 不会，所有处理都在本地完成                                    │
│                                                                  │
│ Q: 刷新页面后照片消失？                                          │
│ A: 正常现象，照片存储在内存中，需要重新上传                      │
│                                                                  │
