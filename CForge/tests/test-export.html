导出功能测试

测试步骤

  1. 打开 index.html
  2. 填写角色基础信息（姓名、职业等）
  3. 在右侧预览面板点击照片上传区域，上传一张测试图片
  4. 测试以下导出功能：
    • 导出 TXT - 应生成带边框的纯文本文件
    • 导出 Markdown - 应生成包含照片的 .md 文件
    • 导出图片 - 应生成预览卡片的 PNG 截图
    • 复制文本 - 应复制纯文本到剪贴板

预期结果

TXT 导出

  • 文件名格式: [角色名]_档案_[时间戳].txt
  • 包含 ═══ 边框装饰
  • 包含所有角色信息（基础、属性、性格、外貌、心理）
  • 属性显示为 ★☆ 星级

Markdown 导出

  • 文件名格式: [角色名]_档案_[时间戳].md
  • 如果上传了照片，应在顶部显示图片（Base64 编码）
  • 使用 Markdown 格式（标题、列表、表格）
  • 属性以表格形式展示

PNG 导出

  • 文件名格式: [角色名]_档案_[时间戳].png
  • 首次使用会自动加载 html2canvas 库
  • 生成预览卡片的高清截图（2x scale）
  • 包含上传的照片（如果有）

复制文本

  • 复制与 TXT 导出相同的格式
  • 显示"已复制到剪贴板"提示
  • 可直接粘贴到任何文本编辑器

功能清单

注意事项

  • PNG 导出首次使用需要加载 html2canvas 库，可能需要几秒钟
  • 照片会以 Base64 格式嵌入 Markdown 文件中
  • 确保浏览器允许下载多个文件
  • 测试时建议使用小尺寸图片（< 1MB）
