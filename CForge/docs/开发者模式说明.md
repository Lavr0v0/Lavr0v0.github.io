# 开发者模式说明

## 什么是开发者模式？

开发者模式是一个特殊功能，允许使用预配置的 API 密钥进行测试和演示。

## 为什么需要开发者模式？

由于程序已部署到公开网页，为了保护 API 密钥不被滥用，默认情况下不再预填充 API 配置。用户需要自己配置 API。

但对于开发者和测试人员，可以通过输入密码启用开发者模式，快速使用预配置的 API。

## 如何启用开发者模式？

### 方法 1: 通过设置面板
1. 点击右上角的齿轮图标打开"API 设置"
2. 在"API Key"标签右侧，点击"开发者模式"按钮
3. 输入密码：`131313`
4. 点击确定

### 方法 2: 直接调用（控制台）
```javascript
// 在浏览器控制台中输入
showDevModePrompt()
```
