**特点**:

## 性能优化

### 使用 GPU 加速
所有变换动画使用 `transform` 和 `opacity`，触发 GPU 加速：
- `transform: translateY()`
- `transform: scale()`
- `transform: rotate()`
- `opacity`

### 避免重排
不使用会触发重排的属性：
- ❌ `width`, `height`
- ❌ `top`, `left`
- ✅ `transform`, `opacity`

### 缓动函数
使用 `cubic-bezier(0.4, 0, 0.2, 1)` 提供自然的缓动效果

## 自定义动画

### 修改动画速度
在 CSS 中搜索 `duration` 或 `transition`，修改时间值：

```css
/* 默认 */
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);

/* 更快 */
transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1);

/* 更慢 */
transition: all 0.5s cubic-bezier(0.4, 0, 0.2, 1);
```

### 禁用动画
如果需要禁用所有动画，添加以下 CSS：

```css
* {
  animation: none !important;
  transition: none !important;
}
```

### 添加新动画
1. 在 `<style>` 标签中定义 `@keyframes`
2. 在元素上应用动画类
3. 设置持续时间和缓动函数

## 动画设计原则

### 1. 微妙而不夸张
动画应该增强体验，而不是分散注意力。

### 2. 快速响应
交互动画应该在 0.3 秒内完成。

### 3. 一致性
相似的元素使用相似的动画。

### 4. 有意义
每个动画都应该有明确的目的。

### 5. 性能优先
使用 GPU 加速，避免重排。

## 常见问题

### Q: 动画太快/太慢？
A: 修改 CSS 中的 `duration` 值。

### Q: 动画卡顿？
A: 检查是否使用了 `transform` 和 `opacity`，避免使用 `width`、`height` 等属性。

### Q: 如何禁用某个动画？
A: 找到对应的 CSS 类，设置 `animation: none` 或 `transition: none`。

### Q: 动画在旧浏览器不工作？
A: 这是正常的降级行为，功能不受影响。
