# Character Forge V7.1 快速开始

## 🚀 5 分钟上手指南

### 第一步：打开工具
直接在浏览器中打开 `index.html`，无需安装任何软件。

### 第二步：配置 API（二选一）

#### 方式 A：开发者模式（快速体验）
1. 点击右上角 ⚙️ 设置
2. 点击"开发者模式"
3. 输入密码：`131313`
4. 完成！

#### 方式 B：使用自己的 API（推荐）
1. 点击右上角 ⚙️ 设置
2. 选择服务商（推荐 DeepSeek）
3. 输入你的 API Key
4. 点击"测试连接"
5. 保存

### 第三步：创建角色（5 步流程）

#### Step 1: 基础档案 📋
填写基础信息：
- 姓名、职业、性别
- 种族、年龄、身高
- 阵营

调整 6 项属性（拖动滑块）：
- 力量、敏捷、体质
- 智力、感知、魅力

#### Step 2: 外貌特征 👁️
选择外貌特征（支持多选）：
- 头发：长度、样式、颜色
- 眼睛：颜色、形状
- 脸型、肤色、体型
- 特殊标记、配饰、服装

💡 提示：选择"疤痕"、"刺青"等特殊标记，Step 4 会询问详细来历

#### Step 3: 心理特征 🧠
调整 8 项性格维度（拖动滑块）：
- 外向 ↔ 内向
- 理性 ↔ 感性
- 谨慎 ↔ 冲动
- 乐观 ↔ 悲观
- 合作 ↔ 竞争
- 传统 ↔ 创新
- 自信 ↔ 谦逊
- 独立 ↔ 依赖

选择心理特征（支持多选）：
- 恐惧、创伤
- 喜好、厌恶
- 社会评价、物品

💡 提示：选择"创伤"、"特殊物品"等，Step 4 会深入探索

#### Step 4: 深度细化 💬
全新的人机协作流程：

```
1️⃣ AI 问问题
   "你选择了刀疤，请描述这些伤疤的具体位置、形状、大小，以及它们的来历。"

2️⃣ 你简要回答
   "五年前在战斗中受伤，左脸颊有一道 5cm 的刀疤"

3️⃣ AI 生成详细内容
   [AI 根据你的回答生成 200-300 字的详细描述]

4️⃣ 你选择修改方式
   ✅ 满意 → 进入下一题
   📝 小改 → 直接修改文本
   💬 大改 → 给 AI 提建议（如："增加更多情感描写"）

5️⃣ 通过，进入下一题
```

💡 提示：
- 简要回答即可，AI 会帮你扩展
- "大改"可以多次使用，直到满意
- 所有内容都会保存到最终报告

#### Step 5: 报告生成 📄
选择报告格式：
- 📖 小说风格
- 🎲 TRPG 卡片
- 📋 档案格式
- 🎤 访谈记录
- 📔 日记体
- 🏥 医疗档案

点击"生成报告"，等待 AI 创作。

### 第四步：导出角色

#### 上传照片 📸
1. 在右侧预览面板顶部
2. 点击虚线框
3. 选择图片（建议 < 1MB）
4. 照片会显示在预览中

#### 导出格式（4 种）

| 格式 | 图标 | 说明 | 适用场景 |
|------|------|------|----------|
| TXT | 📄 | 纯文本，带边框装饰 | 快速分享、打印 |
| Markdown | 📝 | 标准 MD 格式，含照片 | 文档管理、团队协作 |
| PNG | 🖼️ | 高清截图 | 社交媒体、视觉展示 |
| 复制 | 📋 | 复制到剪贴板 | 快速粘贴 |

点击预览面板右上角的对应图标即可导出。

### 第五步：保存和加载

#### 保存角色
1. 点击右上角 💾 导出
2. 选择保存位置
3. 文件名：`[角色名]_[时间戳].json`

#### 加载角色
1. 点击右上角 📂 导入
2. 选择之前保存的 JSON 文件
3. 所有数据自动恢复

## 🎯 使用技巧

### Step 4 最佳实践

#### ✅ 好的简要回答
- "五年前在战斗中受伤"
- "从祖母那里继承的项链"
- "童年时目睹了火灾"

#### ❌ 不好的简要回答
- "不知道"（太模糊）
- "很多很多原因..."（太复杂）
- 留空（AI 无法生成）

#### ✅ 好的修改建议
- "增加更多情感描写"
- "改成发生在童年时期"
- "强调这个物品的神秘感"

#### ❌ 不好的修改建议
- "改一下"（太模糊）
- "不好"（没有方向）
- "重写"（没有具体要求）

### 选择修改方式

| 情况 | 选择 | 说明 |
|------|------|------|
| 内容基本满意，只需微调 | 📝 小改 | 直接修改文本 |
| 需要改变方向或增加内容 | 💬 大改 | 给 AI 提建议 |
| 完全满意 | ✅ 通过 | 进入下一题 |

## 🔧 常见问题

### Q: Step 4 没有生成问题？
A: 需要在 Step 2/3 选择特殊选项：
- Step 2: 疤痕、刺青、特殊标记
- Step 3: 创伤、恐惧、特殊物品

### Q: AI 生成失败？
A: 检查 API 配置：
1. 使用 `tests/test-api.html` 测试连接
2. 确认 API Key 正确
3. 检查网络连接

### Q: 可以多次修改吗？
A: 可以！选择"大改"可以反复生成，直到满意。

### Q: 照片上传失败？
A: 检查：
1. 图片格式（支持 JPG、PNG、GIF、WebP）
2. 图片大小（建议 < 1MB）
3. 浏览器权限

### Q: PNG 导出失败？
A: 首次使用需要加载 html2canvas 库，请等待几秒钟。
